## 1. 执行摘要

Eidon 是一款面向内容创作者的 **AI 视频工作室**，核心提供两项能力：

- **视频翻译 & 配音**：一键将任意平台（YouTube、Bilibili、抖音等）的视频翻译成目标语言，自动生成字幕、AI 配音并输出成品视频。
- **AI 视频创作**：输入主题，AI 自动生成脚本、旁白、配乐，并组装成可直接发布的短视频。

全球短视频内容消费持续爆发，但语言壁垒导致大量优质内容被锁死在单一市场。同时，普通创作者缺乏视频制作专业能力和资金，内容生产成本居高不下。Eidon 用 AI 将原来需要 **翻译团队 + 配音演员 + 剪辑师** 协作完成的流程压缩至 **单人、分钟级、零专业门槛**。

**我们正在寻求 Pre-A 轮融资，目标金额 ¥1,500 万元人民币**，用于产品商业化、云端 SaaS 部署和市场拓展。

## 2. 问题与市场机会

### 2.1 语言壁垒锁死内容价值

- 全球每分钟上传视频超过 500 小时（YouTube 数据）。
- 90% 以上的创作者仅以母语发布内容，单一语言版本意味着放弃了全球 80% 以上的潜在受众。
- 专业视频本地化（翻译 + 配音 + 字幕）的市场报价：¥5,000–¥30,000 / 小时内容，中小创作者和 MCN 机构完全无力承受。

### 2.2 内容生产成本过高

- 一条 3 分钟口播视频的制作成本（脚本 + 拍摄 + 剪辑 + 配乐）行业均价 ¥2,000–¥8,000。
- 小红书、抖音、视频号每日需要大量"知识型短视频"，内容供给严重不足。
- 单人创作者/素人博主面临从 0 到 1 的技能门槛和设备成本。

### 2.3 现有工具的痛点

| 痛点 | 现有工具的局限 |
|------|--------------|
| 视频翻译 | 字幕翻译质量差，无配音，需手动操作多个工具 |
| 视频创作 | 文生视频成本高（Sora/Gen-3）、素材受限、学习曲线陡 |
| 流程整合 | 各工具孤立，Whisper + FFmpeg + LLM 的集成需要技术能力 |
| 隐私/成本 | 全云端方案按次计费，内容上传有隐私风险 |

## 3. 产品与解决方案

### 3.1 两大核心流程

**翻译配音流水线**
```
视频链接 / 本地文件
    ↓ yt-dlp 下载
    ↓ Whisper ASR 转录
    ↓ LLM 批量翻译字幕
    ↓ Edge TTS / CosyVoice2 语音合成
    ↓ FFmpeg 混音 + 字幕烧录
    → 成品视频（可直接发布）
```

**AI 创作流水线**
```
主题 + 时长设定
    ↓ LLM 生成分镜脚本（JSON 结构化）
    ↓ TTS 生成旁白音频
    ↓ 本地/Stock 素材按场景拼接
    ↓ BGM 混音 + FFmpeg 合成
    → 成品短视频
```

### 3.2 关键差异化能力

- **实时进度推送**：WebSocket 每一步实时可见，不是黑箱等待。
- **LLM 热配置**：支持 OpenAI / DeepSeek / Qwen / 任意 OpenAI 兼容接口，无厂商锁定。
- **声音克隆（可选）**：集成 CosyVoice2 本地语音克隆，输出与原声高度近似的配音。
- **数字人口型同步（可选）**：集成 HeyGem，视频人物口型与配音同步。
- **自托管 / SaaS 双模式**：开发者自部署，或企业使用我们托管的云端版本。

### 3.3 当前产品状态

- MVP 已完成：两条流水线完整跑通，前端 UI 完整，WebSocket 实时更新运行稳定。
- 支持 10 种语言（中、英、日、韩、西、法、德、俄、葡、阿）。
- 支持全平台视频来源（YouTube、Bilibili、抖音等 yt-dlp 覆盖的所有站点）。
- 本地化部署，数据不离境，满足企业数据合规需求。

## 4. 技术壁垒

### 4.1 流水线工程积累

将 Whisper、LLM、TTS、FFmpeg 等异构模型和工具整合进一条低延迟异步流水线，并处理好时间码对齐、音频与视频节奏匹配、字幕断句等大量工程细节，是显著的执行壁垒。

### 4.2 多模型组合灵活性

系统设计允许替换任意环节的底层模型（TTS 引擎、LLM 厂商、ASR 模型大小），使产品能跟随模型快速迭代而不需要重构。这保证了我们持续保持在最优性价比的模型组合上。

### 4.3 数据飞轮（规划中）

随着 SaaS 版本积累用户使用数据，可以：
- 持续优化翻译字幕的断句和情感匹配质量。
- 训练针对创作者风格的声音克隆模型。
- 建立行业特化的 LLM 提示模板库（科技、财经、娱乐等）。

### 4.4 私有化部署能力

在数据合规要求严格的企业市场（广电、教育、政务），纯本地化方案是必要条件，而非锦上添花。这一能力本身是竞争壁垒。

## 6. 市场规模

### 6.1 全球视频本地化市场

- 2025 年全球视频本地化市场规模约 **80 亿美元**，年复合增长率 14%。
- AI 驱动的视频翻译/配音解决方案是增速最快的子市场，预计 2028 年市场规模超过 **200 亿美元**。

### 6.2 中国内容创作者市场（重点切入）

- 中国短视频创作者（有变现行为）规模超过 **3,000 万人**。
- 内容出海需求快速增长：抖音海外版（TikTok）、小红书、视频号国际化，催生大量翻译+配音需求。
- MCN 机构超过 **2 万家**，均需持续的视频本地化产能。

### 6.3 TAM / SAM / SOM

| 层级 | 定义 | 规模估算 |
|------|------|---------|
| TAM（总可寻址市场） | 全球视频 AI 本地化 + AI 内容创作工具 | ¥1,400 亿人民币 |
| SAM（可服务市场） | 中文语境内容创作者 + 出海 MCN | ¥120 亿人民币 |

## 7. 竞争分析

### 7.1 竞争格局

| 竞品 | 定位 | 局限 |
|------|------|------|
| HeyGen | 全球领先数字人视频平台 | 定价高（$29–$89/月），数字人锁定，无开源版 |
| Rask AI | 专注视频翻译配音 | 海外产品，中文体验差，无私有化选项 |
| 剪映 AI 翻译 | 字幕翻译，字节系 | 无 AI 配音，流水线不完整，数据在字节 |
| Sora / Pika | AI 文生视频 | 成本高，素材受限，无翻译能力 |
| 自搭 Whisper + FFmpeg | 技术用户 DIY | 无 UI，无流程整合，需要大量工程工作 |

### 7.2 Eidon 的差异化定位

```
          高定制 / 私有化
               ↑
   [企业私有化]  |   [Eidon 企业版]
               |
低价 ←---------+--------→ 高价
               |
   [剪映]       |   [HeyGen / Rask]
               ↓
           纯 SaaS / 闭源
```

Eidon 的核心差异：
1. **唯一同时覆盖翻译配音 + AI 创作的一站式工具**。
2. **开源社区版建立开发者信任**，企业版转化自然。
3. **支持本地 LLM**（DeepSeek、Qwen），规避数据出境合规风险，中国企业客户首选。
4. **定价比 HeyGen 低 60–80%**，功能覆盖更广。
